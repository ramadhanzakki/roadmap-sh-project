# Testimonial Card Mini Project

This is a simple testimonial card project built using HTML and CSS only. It is designed to replicate a clean, modern testimonial section like the one shown in the provided reference image.

## 🔍 Project Overview

- `index.html` contains the page structure with:
  - A testimonial title section
  - A card container holding three testimonial cards
  - Each card includes a user image, name, role, and quote text
- `style.css` provides all visual styling, including:
  - responsive layout for desktop and mobile
  - gradient background and soft shadows
  - floating profile images on top of cards
  - hover transform effect for cards

## 🎨 Features

- Centered title and subtitle
- Card-based testimonial layout
- Image avatar circle effect
- Smooth hover animation
- Responsive design for smaller screens

## 🚀 How to Run

1. Open `index.html` in a web browser.
2. Ensure `style.css` is correctly linked.
3. Make sure the image files in the `images/` folder match the sources in HTML.

## 🛠️ Customization

- Change user names, roles, and quotes inside `index.html`.
- Replace images in the `images/` folder with your own photos.
- Tune color palette and spacing in `style.css`.
